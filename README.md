# Analyzing-E-Learning-Platform-Purchases-using-MySQL

## Project Overview

This project analyzes purchase data from an online e-learning platform using MySQL.

The platform sells digital courses across multiple categories such as:
- Programming
- Data Science
- Marketing

The project focuses on:
- Database creation
- Table relationships
- SQL joins
- Analytical queries
- Business insights

The analysis helps understand:
- Learner purchasing behavior
- Revenue trends
- Popular courses
- Category performance
