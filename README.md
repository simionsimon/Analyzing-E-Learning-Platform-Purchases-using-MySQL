E-Learning Platform Purchases Analysis using MySQL
📌 Project Overview

This project demonstrates how MySQL can be used to analyze sales data from an E-Learning Platform. The database stores information about learners, courses, and purchases, enabling meaningful business insights through SQL queries. The project focuses on relational database design, SQL joins, aggregations, and analytical reporting to support data-driven decision-making.

🎯 Project Objectives
Design a relational database for an online learning platform.
Store learner, course, and purchase information.
Perform SQL joins to combine related data.
Analyze learner purchasing behavior.
Identify top-selling courses and revenue-generating categories.
Generate business insights using SQL analytical queries.
🛠️ Technologies Used
Database: MySQL
Language: SQL
IDE: MySQL Workbench / VS Code
Concepts: Relational Database Management System (RDBMS)
📂 Database Schema

The project contains three related tables:

👨‍🎓 Learners
Column	Description
learner_id	Primary Key
full_name	Learner Name
country	Country
📚 Courses
Column	Description
course_id	Primary Key
course_name	Course Name
category	Course Category
unit_price	Price per Course
🛒 Purchases
Column	Description
purchase_id	Primary Key
learner_id	Foreign Key
course_id	Foreign Key
quantity	Quantity Purchased
purchase_date	Purchase Date

The schema links learners to courses through purchases using foreign keys.

📊 Analytical SQL Queries

The project answers several business questions:

1️⃣ Learner Spending Analysis
Total spending by each learner
Country-wise spending
2️⃣ Top Purchased Courses
Most purchased courses
Revenue generated by each course
3️⃣ Category Performance
Revenue by course category
Number of unique learners
4️⃣ Multi-Category Learners
Learners who purchased courses from multiple categories
5️⃣ Unsold Courses
Courses with no purchases

These queries use aggregate functions, grouping, sorting, and joins to derive actionable insights.

📈 Business Insights

The analysis highlights several findings:

📚 Data Science is the highest revenue-generating category.
🐍 Python for Beginners is among the most popular courses.
🌍 Learners from multiple countries contribute to sales.
💡 Learners purchasing across categories are highly engaged.
📉 Some courses remain unsold and may require promotional efforts.
